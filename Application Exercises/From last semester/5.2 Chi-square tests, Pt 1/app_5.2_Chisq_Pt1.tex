\section*{Pick a number, any number...}

Pick a number between 0 and 9, and submit using your clicker. We will use these data to evaluate whether each number is equally likely to be chosen.

\begin{enumerate}

\item What type of test is most appropriate? Explain your reasoning.

\item What are the hypotheses?

\item Calculate the expected counts.

\item Check if the conditions for inference are met.

\item Calculate the test statistic.

\item Calculate the p-value.

\item What is the conclusion of the hypothesis test?

\end{enumerate}
